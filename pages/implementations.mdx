### Commonmeta-py

Commonmeta-py is a Python library to implement Commonmeta, the common Metadata model for Scholarly Metadata. Use commonmeta-py to convert scholarly metadata, in a variety of formats, listed below. Commonmeta-py is work in progress, the first release on PyPi (version 0.5.0) was on February 16, 2023.
